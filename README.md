# SMR Audio Production Tool

Next.js app for MIT Sloan Management Review audio workflow:

- **Script Prep**: Upload article source HTML and generate a clean TTS-ready script.
- **Teaser Trimmer**: Upload full narration audio and export a hard-cut first 60 seconds as WAV.

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

1. Push this folder to a GitHub repository.
2. In Vercel, import the repository.
3. Deploy with default Next.js settings.
4. No environment variables required.
